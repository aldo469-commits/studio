# **App Name**: EJA TransGlobal

## Core Features:

- Página de Inicio Informativa: Presenta una visión general de EJA GlobalTrans, destacando sus servicios principales y propuesta de valor en español.
- Localizador de Oficinas: Un mapa interactivo que muestra todas las ubicaciones de las oficinas de EJA GlobalTrans a nivel global.
- Catálogo de Servicios Detallado: Describe exhaustivamente cada servicio ofrecido, incluyendo transporte terrestre, marítimo y aéreo, logística, y soluciones personalizadas, todo en español.
- Sección 'Acerca De Nosotros': Cuenta la historia de EJA GlobalTrans, su misión, visión y valores, con un enfoque en su experiencia y compromiso con la calidad, escrito en español.
- Formulario de Contacto Inteligente: Un formulario de contacto optimizado para facilitar la comunicación con los clientes, con campos para consultas generales, soporte técnico y solicitudes de presupuesto, con soporte completo para el idioma español.
- Sistema de Cotizaciones y Presupuestos: Permite a los usuarios solicitar cotizaciones y presupuestos para sus necesidades de transporte y logística, proporcionando detalles sobre el tipo de carga, origen, destino y plazos de entrega.
- Herramienta de Seguimiento de Envíos: Un sistema de seguimiento en tiempo real que permite a los clientes rastrear el estado de sus envíos y mercancías en cualquier momento y lugar. It will also make suggestions to the user using generative AI.

## Style Guidelines:

- Color primario: Azul profundo (#1E3A8A), que representa la confianza, la estabilidad y la experiencia en el sector logístico.
- Color de fondo: Gris claro (#F0F4F8), proporcionando un aspecto limpio y profesional.
- Color de acento: Naranja brillante (#F97316) para llamadas a la acción (CTA) y elementos destacados, simbolizando energía y dinamismo.
- Fuente para el cuerpo del texto: 'Inter', una sans-serif moderna y legible para una experiencia de usuario clara y eficiente.
- Fuente para los títulos: 'Space Grotesk', una sans-serif que combina modernidad con un toque tecnológico.
- Utilizar iconos claros y representativos para cada sección y servicio, facilitando la navegación y la comprensión.
- Diseño limpio y estructurado, con un enfoque en la accesibilidad y la facilidad de uso en todos los dispositivos.
- Animaciones sutiles y transiciones suaves para mejorar la interactividad y proporcionar una experiencia de usuario atractiva.
